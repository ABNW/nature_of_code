Phi = N * 137.5 degrees

137.5 degrees is a great angle apparently.

r = c * sqRt of n

Why do we need an angle and an r?

Usually we would place a point on a cartesian co-ordinate space by using an x value and a y value to make a vector.

However, the phyllotaxis math gives a radius and an angle from the 0,0 origin point.

We can use polar to cartesian co-ordinate transformation to take phi and r and transpose it into a cartesian point.

To do this we are going to use cosine.

If we do cosine of phi, it equals adjacent over hypotoneuse.
