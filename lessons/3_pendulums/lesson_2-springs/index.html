Springs

Rules of a spring

Modeled using Hooks law...

As the extension, so the force.

The force of a spring is directly proportional to the extension of the spring.

The more we contract or extend the spring, the stronger the force, unless at rest, a resting spring has no force

Formula for a spring...

F = -k * x

k is a constant

x is the displacement (either the contraction or extension)

A spring has a rest length. As soon as we displace it from those positions, it will start springing...
